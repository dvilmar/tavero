# Tavero Color Palettes

Paste any of these `colors` blocks into `tailwind.config.js` → `theme.extend.colors`.

---

## 1. Stone + Amber (current — warm & rustic)

```js
primary:     '#1C1917',  // stone-900
primaryLight:'#292524',  // stone-800
accent:      '#D97706',  // amber-600
accentSoft:  '#FEF3C7',  // amber-100
surface:     '#FFFFFF',
background:  '#FAFAF9',  // stone-50
muted:       '#78716C',  // stone-500
mutedLight:  '#A8A29E',  // stone-400
border:      '#E7E5E4',  // stone-200
borderSoft:  '#F5F5F4',  // stone-100
danger:      '#DC2626',
success:     '#16A34A',
```
